# FZ-1 OLED IDC Adapter v2

This folder contains Deftaudio's FZ-1 IDC connector adapter PCB design for the Pico OLED installation.

## Contents

- `fz1_oled_IDC_adapter.kicad_pro` - KiCad project.
- `fz1_oled_IDC_adapter.kicad_sch` - KiCad schematic.
- `fz1_oled_IDC_adapter.kicad_pcb` - KiCad PCB layout.
- `fz1_oled_IDC_adapter_v2_gerbers.zip` - Ready-to-upload Gerber/drill package for PCB manufacture.
- `export1.0/` - Loose Gerber and drill files.
- `libs/` - Local KiCad symbol/footprint support files included with the project.

Upload `fz1_oled_IDC_adapter_v2_gerbers.zip` to a PCB manufacturer such as PCBWay or a similar service.
