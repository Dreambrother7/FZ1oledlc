# FZ-10M / FZ-20M / VZ-10M OLED IDC Adapter

This folder contains Deftaudio's FZ-10M, FZ-20M, and VZ-10M IDC connector adapter PCB design for the Pico OLED installation.

## Contents

- `fz10_oled_IDC_adapter.kicad_pro` - KiCad project.
- `fz10_oled_IDC_adapter.kicad_sch` - KiCad schematic.
- `fz10_oled_IDC_adapter.kicad_pcb` - KiCad PCB layout.
- `fz10m_fz20m_vz10m_oled_IDC_adapter_gerbers.zip` - Ready-to-upload Gerber/drill package for PCB manufacture.
- `export1.0/` - Loose Gerber and drill files.
- `libs/` - Local KiCad symbol/footprint support files included with the project.

Upload `fz10m_fz20m_vz10m_oled_IDC_adapter_gerbers.zip` to a PCB manufacturer such as PCBWay or a similar service.
